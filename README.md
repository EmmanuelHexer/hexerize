## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/EmmanuelHexer/hexerize.git
   cd hexerize
````

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start development server:

   ```bash
   npm run dev
   ```
